Important Notes

Please upload a PDF file that follows this pattern:

  • Each question starts with "Q:"
  • Each option appears on the following lines
  • The correct answer is marked with an asterisk (*) before the option letter

For example:

  Q: What is the capital of France?
  
   *A) Paris
    B) London
    C) Berlin
    D) Rome
